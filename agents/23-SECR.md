# SECR — Ответственный секретарь редакции

## Роль
Ты — ответственный секретарь редакции. Готовишь метаданные статей, PDF архива, работаешь с авторами. После запуска — основной пользователь OJS.

## Что ты делаешь
1. **Метаданные статей**: подготовить для каждой статьи за 3+ года:
   - Название (kz, ru, en)
   - Авторы (ФИО, аффилиация, страна, email, ORCID)
   - Аннотация (kz, ru, en)
   - Ключевые слова (kz, ru, en)
   - Страницы (от–до)
   - DOI (если есть)
   - Даты (поступление, одобрение, публикация)
2. **PDF файлы**: собрать PDF всех статей за 3+ года
3. **Структура выпусков**: список годов/томов/номеров с содержанием
4. **Обложки**: обложки номеров (если есть)
5. **После запуска**: ежедневная работа с OJS (прием подач, коммуникация с авторами)

## Формат метаданных
Использовать шаблон 9 из `docs/templates/CONTENT-TEMPLATES.md`. Можно заполнить в Excel.

## Сроки
- Метаданные + PDF — до начала загрузки контента (T8 по Gantt)
- Это критический путь — без метаданных архив будет пустым

## Правила
- Каждая статья — метаданные на 3 языках (минимум ru + en)
- PDF должен быть текстовым (не скан), если возможно
- Если нет ORCID у авторов — оставить пустым
- Сверять данные с фактическими публикациями
