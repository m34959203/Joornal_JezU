## 1. Бизнес-обоснование

Жезказганский университет им. О.А. Байконурова издаёт научный журнал «Вестник ЖезУ» с 2004 года. Для включения журнала в **Перечень изданий ККСОН** (Список 2 или Список 3) необходимо обеспечить информационную открытость через собственный веб-сайт, соответствующий требованиям Приказа МОН РК №20 (ред. 03.06.2025 №289).

Отсутствие сайта **блокирует** возможность включения в Перечень и снижает привлекательность журнала для авторов.

## 2. Цели проекта

| # | Цель | Измеримый результат |
|---|------|---------------------|
| C1 | Обеспечить информационную открытость журнала | Сайт с полной информацией на 3 языках (kz/ru/en) |
| C2 | Реализовать онлайн-систему подачи и рецензирования | Полный цикл: подача → рецензирование → решение → публикация |
| C3 | Соответствовать требованиям ККСОН (пункт 5/6, подпункт 15) | Чек-лист соответствия 100% обязательных требований |
| C4 | Обеспечить DOI для каждой статьи | Интеграция с CrossRef, DOI присвоен всем статьям |
| C5 | Загрузить архив номеров | Минимум 3 года архива с метаданными и PDF |

## 3. Scope (Границы проекта)

### В scope:
- Развертывание и настройка OJS 3.4.x
- Разработка уникальной темы оформления (child theme)
- Настройка 3 языков (казахский, русский, английский)
- Настройка ролей, форм подачи, рецензирования
- Интеграция плагинов: DOI/CrossRef, ORCID, статистика
- Анализ интеграции со StrikePlagiarism.com
- Загрузка архива номеров (при предоставлении материалов заказчиком)
- Обучение редакции (инструкция + консультации)
- Тестирование и ввод в эксплуатацию

### Вне scope:
- Написание контента (редполитика, этика, руководства) — ответственность заказчика
- Регистрация домена и оплата хостинга — ответственность заказчика
- Получение аккаунта CrossRef и оплата DOI — ответственность заказчика
- Получение Client ID/Secret ORCID — ответственность заказчика (при консультации ЦИИ)
- Перевод текстов на казахский и английский — ответственность заказчика

## 5. Ограничения

- **Сроки**: 15 рабочих дней (дизайн 5 + верстка 5 + тестирование 5)
- **Бюджет**: в рамках внутреннего финансирования ЦИИ
- **Технологии**: OJS 3.x (требование ТЗ), PHP, MySQL
- **Зависимость от заказчика**: контент, домен, CrossRef аккаунт

## 6. Допущения

- Заказчик предоставит контент параллельно с разработкой
- Сервер ЖезУ (или внешний хостинг) будет выделен до начала этапа верстки
- Архив номеров за 3+ года существует в виде PDF-файлов
- Домен будет зарегистрирован/выделен до этапа тестирования

## 7. Риски верхнего уровня

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Заказчик не предоставляет контент вовремя | Высокая | Высокое | Еженедельный контроль, шаблоны контента |
| Домен/хостинг не готов к моменту деплоя | Средняя | Высокое | Развертывание на временном сервере ЦИИ |
| Нет аккаунта CrossRef для DOI | Средняя | Среднее | Настройка плагина с тестовыми DOI |
| StrikePlagiarism не имеет открытого API | Средняя | Низкое | Описание ручного workflow |

## 8. Критерии приемки

1. Сайт доступен по согласованному домену через HTTPS
2. Все разделы меню функционируют на 3 языках
3. Онлайн-подача статьи проходит полный цикл (от регистрации до решения)
4. Архив номеров за 3+ года загружен и доступен
5. Поиск работает по названиям, авторам, ключевым словам, DOI
6. DOI ссылки корректно ведут на страницу статьи
7. Сайт адаптивен (мобильные + десктоп)
8. Время загрузки главной страницы ≤ 3 секунд
9. Редакция обучена работе с системой
