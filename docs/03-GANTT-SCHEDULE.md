## Критический путь

```
Подписание ТЗ → Развертывание OJS → Настройка журнала → Child Theme →
→ Загрузка контента → Функциональное тестирование → Демонстрация → Приемка → Запуск
```

**Узкое место**: получение контента от заказчика (Фаза 3.1). Если контент задерживается, задерживается загрузка (3.2) и тестирование (4.5).
