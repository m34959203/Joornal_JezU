# Матрица ответственности (RACI)

**Проект:** JRNL-2026 | **Версия:** 1.0 | **Дата:** 2026-03-23

---

## Обозначения

| Символ | Значение | Описание |
|--------|----------|----------|
| **R** | Responsible (Исполнитель) | Делает работу |
| **A** | Accountable (Ответственный) | Утверждает результат, несет ответственность |
| **C** | Consulted (Консультант) | Дает экспертное мнение до выполнения |
| **I** | Informed (Информируемый) | Получает информацию о результате |
