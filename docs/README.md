## Быстрый старт для команды

1. Прочитать [Паспорт проекта](01-PROJECT-CHARTER.md) — понять scope и ограничения
2. Изучить [WBS](02-WBS-TASKS.md) — найти свои задачи
3. Проверить [RACI](05-RACI-MATRIX.md) — понять зону ответственности
4. Для разработчиков: [Техническая спецификация](07-TECH-SPEC.md) + [ADR](architecture/ADR-001-PLATFORM.md)
5. Для заказчика: [Шаблоны контента](templates/CONTENT-TEMPLATES.md) + [Чек-лист ККСОН](06-KKSON-COMPLIANCE.md)
