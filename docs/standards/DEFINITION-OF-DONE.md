# Definition of Done (Критерии завершенности)

**Проект:** JRNL-2026 | **Версия:** 1.0 | **Дата:** 2026-03-23

---

## Общие критерии (для любой задачи)

- [ ] Код/конфигурация зафиксированы в git
- [ ] Изменения проверены на staging-среде
- [ ] Нет критических ошибок (блокирующих работу пользователя)
- [ ] Функционал работает на 3 языках (kz, ru, en) — если применимо
- [ ] Адаптивная верстка проверена (десктоп + мобильный)

---

## По фазам

### Фаза 1: Дизайн — Done когда:
- [ ] Макеты всех страниц готовы в Figma (десктоп + мобайл)
- [ ] Цветовая палитра и типографика задокументированы
- [ ] Макеты утверждены заказчиком (подпись/email-подтверждение)
- [ ] Не более 2 итераций правок

### Фаза 2: Разработка — Done когда:

**2.1 Развертывание OJS:**
- [ ] OJS доступен по HTTPS
- [ ] SSL-сертификат валиден
- [ ] SMTP отправляет email
- [ ] Cron-задачи OJS настроены и работают
- [ ] Автоматический бэкап настроен (БД + файлы, ежедневно)
- [ ] Docker Compose файл задокументирован

**2.2 Настройка журнала:**
- [ ] Журнал создан, основные настройки заполнены
- [ ] 3 языка настроены, русский — основной
- [ ] Роли: автор, рецензент, редактор, администратор — функционируют
- [ ] Форма подачи статьи содержит все обязательные поля (название, аннотация, ключевые слова, авторы, файл — на 3 языках)
- [ ] Workflow рецензирования: подача → назначение → рецензия → решение
- [ ] Email-уведомления отправляются на каждом этапе
- [ ] Форма рецензирования содержит оценки и комментарии

**2.3 Child Theme:**
- [ ] Шапка: логотип, название, меню, поиск, языки, вход, ISSN
- [ ] Подвал: эмблема, учредитель, адрес, «основан в 2004», метрика
- [ ] Главная: текущий номер, обложка, кнопки CTA
- [ ] «О журнале»: боковые кнопки (цели, политика, коллегия, этика)
- [ ] «Архив»: навигация год → номер → статьи
- [ ] «Редколлегия»: таблица 3 колонки + кнопки Scopus/WoS/ORCID
- [ ] Страница статьи: метаданные, DOI, скачивание PDF
- [ ] Результаты поиска: название, авторы, выходные данные, ссылка
- [ ] Хлебные крошки на внутренних страницах
- [ ] Кнопки CTA: «ПОДАТЬ СТАТЬЮ», «ТЕКУЩИЙ НОМЕР»
- [ ] Адаптивность: 1920px, 1366px, 768px, 375px
- [ ] Время загрузки главной ≤ 3 секунд

**2.4 Плагины:**
- [ ] DOI/CrossRef: плагин установлен, настроен (тестовый или реальный)
- [ ] ORCID: плагин установлен, отображается на странице статьи
- [ ] Аналитика: Яндекс.Метрика или GA подключена
- [ ] Usage Statistics: просмотры/загрузки считаются
- [ ] StrikePlagiarism: анализ проведен, результат задокументирован

**2.5 Поиск:**
- [ ] Поиск по названиям статей — работает
- [ ] Поиск по авторам — работает
- [ ] Поиск по ключевым словам — работает
- [ ] Поиск по DOI — работает
- [ ] Пустые результаты: «По вашему запросу ничего не найдено»

### Фаза 3: Контент — Done когда:
- [ ] Все статические страницы заполнены на 3 языках
- [ ] Редколлегия загружена с ORCID/Scopus/WoS ссылками
- [ ] Архив за 3+ года: выпуски созданы, метаданные + PDF загружены
- [ ] DOI присвоены всем статьям
- [ ] Текущий номер отображается на главной

### Фаза 4: Тестирование — Done когда:
- [ ] Все тест-кейсы из WBS (4.1.1–4.1.13) пройдены
- [ ] Кроссбраузерность: Chrome, Firefox, Safari, Edge — OK
- [ ] Адаптивность: 4 разрешения проверены
- [ ] Время загрузки ≤ 3 сек
- [ ] HTTPS работает без ошибок
- [ ] Нет грамматических/стилистических ошибок на публичных страницах
- [ ] Бэкап восстановлен успешно (тест)

### Фаза 5: Приемка — Done когда:
- [ ] Заказчик подтвердил соответствие ТЗ
- [ ] Все замечания заказчика устранены
- [ ] Акт сдачи-приемки подписан
- [ ] Главный редактор и секретарь обучены
- [ ] Инструкция передана
- [ ] Логины/пароли переданы
- [ ] Сайт доступен на production-домене

## Критерии качества кода/конфигурации

| Критерий | Требование |
|----------|-----------|
| Версионирование | Все изменения в git (GitHub: m34959203/Joornal_JezU) |
| Секреты | .env файлы в .gitignore, пароли не в коде |
| Docker | docker-compose.yml с документированными переменными |
| Тема OJS | Child theme в отдельной директории, документированная |
| Бэкапы | Ежедневно, хранение 30 дней, тест восстановления |
