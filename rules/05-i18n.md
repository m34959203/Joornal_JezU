# Правила многоязычности (i18n)

## Языки проекта
| Код | Язык | Роль | Полнота локали OJS |
|-----|------|------|--------------------|
| `ru_RU` | Русский | Основной | 100% (официальная) |
| `en_US` | Английский | Обязательный | 100% (официальная) |
| `kk` | Казахский | Обязательный | ~20% (требуется перевод) |

## Правила для шаблонов
- НИКОГДА не хардкодить текст в `.tpl` файлах
- Все строки — через `{translate key="plugins.themes.zhezujournal.keyName"}`
- Ключи именовать: `plugins.themes.zhezujournal.<раздел>.<элемент>`

Примеры ключей:
```
plugins.themes.zhezujournal.header.submitArticle = Подать статью
plugins.themes.zhezujournal.header.currentIssue = Текущий номер
plugins.themes.zhezujournal.footer.foundedYear = Журнал основан в 2004 году
plugins.themes.zhezujournal.search.noResults = По вашему запросу ничего не найдено
plugins.themes.zhezujournal.editorial.chiefEditor = Главный редактор
plugins.themes.zhezujournal.editorial.secretary = Ответственный секретарь
plugins.themes.zhezujournal.editorial.members = Члены редакционной коллегии
```

## Файлы локалей
```
theme/locale/
├── ru_RU/locale.po    # Русский (основной — заполняется первым)
├── en_US/locale.po    # Английский
└── kk/locale.po       # Казахский
```

## Метаданные статей
- При подаче автор заполняет на 3 языках:
  - Название статьи
  - Аннотация
  - Ключевые слова
- Информация об авторах — на языке публикации + английский
- OJS хранит и экспортирует метаданные на всех языках

## Казахская локаль — приоритеты перевода
1. **Критический**: меню, шапка, подвал, кнопки навигации
2. **Высокий**: формы регистрации/входа, формы подачи статьи
3. **Средний**: уведомления, сообщения об ошибках
4. **Низкий**: админка OJS (редакторы обычно работают на русском)

## Порядок работы с переводами
1. Разработчик добавляет ключ в `ru_RU/locale.po`
2. Переводчик KK добавляет перевод в `kk/locale.po`
3. Переводчик EN добавляет перевод в `en_US/locale.po`
4. Коммит: `locale(kk): перевести ключи навигации`

## Проверка
- Переключить язык на каждый из 3 — проверить отображение
- Проверить что нет «????» или пустых строк
- Проверить что кириллица корректно отображается в URL и поиске
